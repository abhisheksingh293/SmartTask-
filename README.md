# SmartTask Hub

SmartTask Hub is a full-stack task management application built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- User Authentication (JWT)
- Task Management (CRUD)
- Responsive UI (Tailwind CSS)

## Tech Stack
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Frontend:** React, Vite, Tailwind CSS

## Getting Started

### Backend
1. `cd server` (or root depending on setup)
2. `npm install`
3. Set up `.env`
4. `npm run dev`

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev`
